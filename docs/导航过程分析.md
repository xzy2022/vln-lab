# 导航过程分析

本文档记录 SAME 实验后处理的常用入口：先跑主实验生成 `eval_items`，再生成细粒度表格，最后按某个 `internal_item_id` 渲染逐样本导航过程图像。

## 1. 主实验

```bash
python scripts/experiments/run_same.py --config configs/same/val_r2r_eval_only.yaml
python scripts/experiments/run_same.py --config configs/same/val_r2r_reverie_cvdn_soon.yaml
```

如果要一次覆盖 R2R/REVERIE/CVDN/SOON 的 seen、unseen 等 split，可使用：

```bash
python scripts/experiments/run_same.py --config configs/same/val_all_r2r_reverie_cvdn_soon.yaml
```

主实验完成后，逐样本输入位于：

```text
experiment_outputs/<experiment_id>/eval_items/*_eval_items.jsonl
```

例如：

```text
experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4/eval_items/R2R_val_seen_eval_items.jsonl
```

其中 `identity.internal_item_id` 是后续逐样本分析的稳定 ID，例如 `r2r_5593_0`。

## 2. 细粒度分析

细粒度指标从 `eval_items` sidecar 生成，不重新跑模型。最常用命令如下：

```bash
python scripts/analysis/build_same_fine_metrics.py \
  --experiment-dir experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4 \
  --connectivity-dir data/same/simulator/connectivity
```

如果需要把结果写到自定义目录：

```bash
python scripts/analysis/build_same_fine_metrics.py \
  --experiment-dir experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4 \
  --connectivity-dir data/same/simulator/connectivity \
  --output-dir reports/item_metrics/0013_same_fine_metrics
```

默认产物位于：

```text
experiment_outputs/<experiment_id>/fine_metrics/
  manifest.json
  jsonl/*_fine_metrics.jsonl
  tables/fine_metrics_wide.csv
  tables/fine_metrics_long.csv
  tables/fine_metrics_summary.json
```

常见筛选方式：

```bash
python - <<'PY'
import csv
from pathlib import Path

wide_csv = Path("experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4/fine_metrics/tables/fine_metrics_wide.csv")

with wide_csv.open("r", encoding="utf-8") as f:
    rows = csv.DictReader(f)
    hard_cases = [
        row for row in rows
        if row["dataset"] == "R2R"
        and row["split"] == "val_seen"
        and row["official.final_success"] == "false"
    ]

for row in hard_cases[:20]:
    print(
        row["internal_item_id"],
        "final_dist=", row["official.final_distance_to_goal_m"],
        "oracle=", row["official.oracle_success"],
        "path_len=", row["common.path_length_m"],
    )
PY
```

更多字段含义见 `docs/fine_metrics.md`。

## 3. 逐样本图片输出

现在逐样本分析可以直接用脚本完成。给定 `internal_item_id`，脚本会自动在实验目录下查找对应 `eval_items`、读取 `scan`、渲染 trajectory 中每个 viewpoint 的 36 视角 contact sheet，并保存 `metadata.json`。

```bash
python scripts/analysis/render_same_eval_item.py \
  r2r_5593_0 \
  --experiment-dir experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4
```

默认输出目录：

```text
experiment_outputs/<experiment_id>/item_replays/<internal_item_id>/
  metadata.json
  step_000_<viewpoint>.jpg
  step_001_<viewpoint>.jpg
  ...
```

如果同一个 ID 在多个文件中有歧义，可加 dataset/split：

```bash
python scripts/analysis/render_same_eval_item.py \
  r2r_5593_0 \
  --experiment-dir experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4 \
  --dataset R2R \
  --split val_seen
```

也可以直接指定某个 `eval_items` 文件：

```bash
python scripts/analysis/render_same_eval_item.py \
  r2r_5593_0 \
  --eval-items experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4/eval_items/R2R_val_seen_eval_items.jsonl \
  --output-dir tmp/replay/r2r_5593_0
```

## 4. scan 数据准备

渲染依赖 Matterport skybox 数据。默认读取：

```text
third_party/Matterport3DSimulator/data/v1/scans/<scan_id>/
```

如果目标 scan 尚未下载，可让逐样本脚本自动下载并预处理：

```bash
python scripts/analysis/render_same_eval_item.py \
  r2r_5593_0 \
  --experiment-dir experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4 \
  --download-missing-scan
```

这等价于先按样本 ID 找到 `scan`，然后执行：

```bash
python scripts/setup/download_mp.py \
  -o third_party/Matterport3DSimulator/data \
  --id <scan_id> \
  --type matterport_skybox_images

python scripts/setup/prepare_mp3d_skybox.py \
  --scans-dir third_party/Matterport3DSimulator/data/v1/scans \
  --scan-id <scan_id>
```

如果 scan 数据放在其他位置：

```bash
python scripts/analysis/render_same_eval_item.py \
  r2r_5593_0 \
  --experiment-dir experiment_outputs/0013_same_val_all_r2r_reverie_cvdn_soon_same_s0_v4 \
  --dataset-path /home/xzy/datasets/mp3d-mini/v1/scans \
  --nav-graph-path third_party/Matterport3DSimulator/connectivity \
  --download-missing-scan \
  --download-root /home/xzy/datasets/mp3d-mini
```

说明：该脚本输出的是 viewpoint 级回放，每个 step 是该 viewpoint 的标准 36 视角图。`eval_items` 默认不含逐步 heading/elevation/action trace，因此这不是 SAME 原始动作级重放。
