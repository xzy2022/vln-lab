

## data/same

使用
```bash
python scripts/setup/download_same_r2r_eval.py --source hf-mirror

```

- `R2R/*.json` 是 R2R 标注与评测 split。里面有指令、路径、scan id、viewpoint 等，属于“任务数据”。
- `data/features/img_features/clip_vit-b16_mp3d_hm3d_gibson.hdf5` 是预提取的视觉特征，SAME 推理时真正喂给模型的是它，不是原始 Matterport3D 场景图像。
- `data/simulator/connectivity/*.json`、`data/simulator/mp3d_connectivity_graphs.json`、`data/simulator/mp3d_scanvp_candidates.json` 是导航图结构、节点位置和候选动作信息。它们替代了真实 simulator 的在线图查询。
- `data/pretrain/Attnq_pretrained_ckpt.pt` 是“预训练初始化权重”。模型构建时会先从这里加载基础参数，`vlnbert_init.py` (line 18)。
- `data/ckpt/SAME.pt` 是“最终训练好的官方 checkpoint”。真正评测/恢复时，会再从这里覆盖加载完整任务权重，`base_trainer.py` (line 68) `base_trainer.py` (line 97)。
- 当前SAME子仓库自带的测试配置默认同时要求 pretrain 路径存在，`test.yaml` (line 1)。
